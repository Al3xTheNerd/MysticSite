Permission Level Selector

Permissions

  • All roles have all features of the ones below it.
  • 100: Literally everything, this is reserved for the site developer
  • 90: Manage permissions of those below.
  • 80: Delete Crates
  • 70: Unused, for now.
  • 60: Manage Item Order.
  • 50: Create new Crates and upload Items
  • 40: Upload Item images.
  • 30: Create and manage sets.
  • 20: Set Item Tags.
  • 10: Set Item notes.
  • 0: Nothing at all, the user can login but do nothing else afterwords.
