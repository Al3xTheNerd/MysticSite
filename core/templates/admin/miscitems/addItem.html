Add Miscellaneous Item

  • This page is used to add brand new miscellaneous items (such as plushies, or server event items) to the site. Instructions at the bottom of the page.

Initial Setup

  • You will need the Component Viewer mod on Fabric 1.21.8. This has not been tested on any newer, or older versions.
  • You will need to open the Component Viewer settings, and ensure the following settings are correct:
  • Clipboard>Copy: Item Stack
  • Clipboard>Formatting: JSON
  • Clipboard>Indentation: 4

Adding An Item

  • Once you have the Component Viewer mod setup, you will press Ctrl+C to copy the data of an item.
  • Then you will press Ctrl+V inside the large input on the left side of the screen.
  • From there, you should see the middle box fill up with an item preview, as well as sme inputs on the right side prefilling themselves. If this did not happen, please reach out to AlexTheNerd on Discord and it will be looked into to see if a bug happened, or if a mistake was made.
  • For most tools/armor pieces, the Primary Tag box will fill itself in, however if the item is a Helmet, then it will likely not mark itself, and you will need to do so.
  • From there, you can fill in the tag slots as needed, don't worry about messing it up, they can always be fixed later.
  • The Item Name input is the name that the Discord bot will use, if it is a duplicate of another item it will not be able to be searched. So if needed, change it.
  • The Notes portion is shown when people click on the item, or when it is searched using the item bot on Discord. You can use Discord formatting such as lists, underlining, bold, italics etc. as you please.
  • Press Add to DB! when finished.
