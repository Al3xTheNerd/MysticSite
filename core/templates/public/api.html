Api Documentation

  • All endpoints/relevant files are listed below.

API Errors

  • 0: NO_RESULTS: No results found in database.
  • 1: INVALID_COLUMN: The database column you requested does not exist.
  • Errors can be seen in the 'messages'portion of the API response.
  • Example:{ "data": [ {} ], "messages": [ { "1": "[ITEMINFOPLS] is not a valid column.." }, { "1": "[GIBINFO] is not a valid column.." } ] }

Images

  • To determine what kind of image exists for an item icon, you will need to make sure to include ImageType in your I-INCLUDED-INFO headers to /api/items or /api/miscitems
  • For Item Lore images, you can request that at {{ServerURL}}static/images/{{ServerName}}_Descriptions/[itemid].png
  • For Item Icon images, you can request that at {{ServerURL}}static/images/{{ServerName}}_Icons/[item.id].[item.ImageType]
  • For Misc. Item Lore images, you can request that at {{ServerURL}}static/images/{{ServerName}}_Misc_Descriptions/[itemid].png
  • For Misc. Item Icon images, you can request that at {{ServerURL}}static/images/{{ServerName}}_Misc_Icons/[item.id].[item.ImageType]

{{ServerURL}}api/items

  • This endpoint is used to query the Crate Item database.
  • Format: GET
  • Default Results: id, ItemName
  • I-INCLUDED-INFO valid options: id, CrateID, TagPrimary, TagSecondary, TagTertiary, TagQuaternary, TagQuinary, TagSenary, TagSeptenary, WinPercentage, RarityHuman, RarityHTML, ItemName, ItemNameHTML, Notes, RawData, ItemHuman, ItemHTML, ItemOrder, ConnectedItems, EfficiencyLevel
  • I-INCLUDED-INFO format: id;CrateId;ItemName
  • Make a GET request with the header I-INCLUDED-INFO to get all item information stored in the database.
  • Python Example:import requests headers = { "I-INCLUDED-INFO" : "id;CrateID;ItemName;ImageType" } results = requests.get("{{ServerURL}}api/items", headers=headers)

{{ServerURL}}api/crates

  • This endpoint is used to query the Crate List database.
  • Format: GET
  • Default Results: id, CrateName, ReleaseDate, URLTag, CrateType
  • Make a GET request to get all Crate information stored in the database.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/crates")

{{ServerURL}}api/miscitems

  • This endpoint is used to query the Misc. Item database.
  • Format: GET
  • Default Results: id, ItemName
  • I-INCLUDED-INFO valid options: id, GroupID, ItemName, ItemNameHMTL, Notes, RawData, ItemHuman, ItemHTML, ItemOrder, ImageType
  • I-INCLUDED-INFO format: id;GroupID;ItemName
  • Make a GET request with the header I-INCLUDED-INFO to get all misc. item information stored in the database.
  • Python Example:import requests headers = { "I-INCLUDED-INFO" : "id;GroupID;ItemName;ImageType" } results = requests.get("{{ServerURL}}api/miscitems", headers=headers)

{{ServerURL}}api/miscgroups

  • This endpoint is used to query the Group List database.
  • Format: GET
  • Default Results: id, GroupName, ReleaseDate, URLTag, GroupType, Notes, GroupOrder
  • Make a GET request to get all Crate information stored in the database.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/miscgroups")

{{ServerURL}}api/sets

  • This endpoint is used to query the Set List database.
  • Format: GET
  • Default Results: id, Name, ItemList, Type, SetDescription, SetOrder
  • Make a GET request to get all Set information stored in the database.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/sets")

{{ServerURL}}api/tags

  • This endpoint is used to query the list of tags in use by the database.
  • Format: GET
  • Default Results: List[str]
  • Make a GET request to get all tags stored in the database.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/tags")

{{ServerURL}}api/itemCount

  • This endpoint is used to query the number of Crate Items currently in the database.
  • Format: GET
  • Default Results: int
  • Make a GET request to get the number of items.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/itemCount")

{{ServerURL}}api/vote/set/[server]/[amount]

  • This endpoint is used to set the current vote count on a server.
  • Format: GET
  • Make a GET request to set the number of votes.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/vote/set/Server1/99")

{{ServerURL}}api/vote/get/[server]

  • This endpoint is used to get the current vote count on a server.
  • Format: GET
  • Make a GET request to get the number of votes.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/vote/get/Server1")

{{ServerURL}}api/itemtracker/[code]

  • This endpoint is used to get the list of items from a 6 digit Item Tracker Export code.
  • Format: GET
  • Default Results: List[Item.id]
  • Make a GET request to get the list of items.
  • Python Example:import requests results = requests.get("{{ServerURL}}api/itemtracker/aoo65d")
