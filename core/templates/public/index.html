{{ ServerItemName }} List

  • This website is a complete list of every custom item on {{ ServerName }}, use it to track your item collection, discover new items to collect, or do some pretend gambling!
  • Below is a list of features you can expect to find on the website.

Crates

  • Check what items are in each crate, as well as get the release date of every crate!

Tags

  • Get a list of all items under a given tag, such as a list of all infinite blocks on the server, or all items that give the speed attribute.

Sets

  • Sets are going to either be Armor sets with a set bonus, or groups of items that pair so well together that nearly everybody knows the combination.

Misc. Items

  • Misc. Items are all the custom items that are not directly in a crate, including but not limited to: the 10x key purchase bonuses, furniture items, currency, or event items.

Item Tracker

  • This tool will give you a list of every item on the server, and you can use this to check off which items you already have, at the bottom of the screen it will give you a list of missing items that you can copy/paste directly into Discord for a nicely formatted list.

Inventory Planner

  • You can use this tool to plan out which items you would like to use in your inventory, then copy the code to share with others and show off you're beautiful loadout.

Block Break Speed Calculator

  • This tool is to do the math for you, to see which tools can instamine which blocks, and in which conditions.

Site Stats

  • Ever been curious about the total number of {{ ServerItemName }}? Or perhaps how many infinite blocks exist? You can check that on this screen.
